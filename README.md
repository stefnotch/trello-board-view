# Trello Board View

A fancy Trello Board view with lightning fast searches. [See it in action here](https://stefnotch.github.io/trello-board-view/).

### Prerequisites

- Node & NPM

### Install

```sh
npm install
```

### Usage

##### Develop

```sh
# run dev server at localhost:8080
npm run dev
```

##### Build

```sh
# transpile js for deployment
npm run build
```
